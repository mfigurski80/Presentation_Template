HOW?

Some knowledge of HTML5 and CSS3 is required, but the outline of this website is simple.

The slide itself is set at constant width and height, allowing the user to easily push objects around without worrying about scaling to different screen sizes.
(except phones. This is not mobile friendly at all, don't hope)

JQuery is minimal and simple to understand.

CSS is fully commented to prevent confusion. All the class names clearly indicate the slide.

The HTML of each slide is clearly marked and separate. Slides can be added as needed.

WHY?

I'm not sure why.

Orignally, this was supposed to provide an alternative to wrestling with other presentation possibilities, such as google slides or the slides.com decks, but eventually I've found ways to circumvent these problems. Now, this template has become more of an exercise in JQuery and CSS.

WHAT?

This template allows it's users to create and experience presentations that utilize the full power of HTML5, CSS3, and JQuery.
These include:

  • Responsiveness
  • Animations
  • Stylizations, including fonts
  • ...And everything else that websites can have (click to edit)
